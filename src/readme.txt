Welcome to Santorini


Windows Installation:

1. Extract zip archive

2. Run santorini-launcher

3. Enjoy

__________________________________________

Linux Installation:

1. Extract tarball

2. chmod +x santorini-launcher

3. Run santorini-launcher

4. Enjoy

__________________________________________

Manual Installation:

Windows:

1. Extract zip archive

2. Run bin/install(32/64).bat

3. Run SantoriniLauncher

4. Enjoy

Linux:

1. Extract tarball

2. chmod +x bin/install.sh

3. bash bin/install.sh

4. Run santorini

5. Enjoy
